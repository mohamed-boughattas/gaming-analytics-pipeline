# Dashboard Screenshots

This directory contains screenshots of the project's dashboards.

## Files to Add

1. **dashboard.png** - Screenshot of the Marimo dashboard
   - Run `marimo edit dashboard/gaming_analytics.py`
   - Take a screenshot showing visualizations
   - Save as `dashboard.png`

2. **evidence.png** - Screenshot of the Evidence dashboard
   - Run `cd evidence && npm run dev`
   - Navigate to http://localhost:3000
   - Take a screenshot showing the analytics pages
   - Save as `evidence.png`

## Tips for Good Screenshots

- Capture meaningful visualizations (charts, KPIs, filters)
- Use a clean window without browser chrome if possible
- Ensure resolution is at least 1200x800
- Keep file sizes reasonable (compress PNGs if large)
